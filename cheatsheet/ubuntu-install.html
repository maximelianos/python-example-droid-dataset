Ubuntu install tips

Ubuntu version 22.04 jammy

--- APT ---

- Disable APT auto-updates

sudo nvim /etc/apt/apt.conf.d/20auto-upgrades
APT::Periodic::Update-Package-Lists "0";
APT::Periodic::Unattended-Upgrade "0";
APT::Periodic::Download-Upgradeable-Packages "0";
APT::Periodic::AutocleanInterval "0";

apt-get remove update-manager
https://askubuntu.com/questions/218755/how-to-disable-the-update-manager-popup

- Timeshift for FS snapshots on BTFRS
sudo apt-add-repository -y ppa:teejee2008/ppa && sudo apt update
sudo apt install timeshift

- grub-customizer for reducing GRUB waiting time
sudo add-apt-repository ppa:danielrichter2007/grub-customizer
sudo apt install grub-customizer

- search new packages
apt-cache search ^gimp

- APT install specific version of package
https://itsfoss.com/apt-install-specific-version/

- Add binary to Applications menu
Look at /usr/share/applications/*.desktop and create your own file

---

- VScode change indent in file explorer tree
https://www.ryanchapin.com/vscode-change-indent-for-file-explorer-tree/

- VScode disable updates
ctrl+shift+p -> user settings (json)
"update.mode": "none"

- VSCode preview markdown by default
ctrl+shift+p -> user settings (json)
{
    "workbench.editorAssociations": {
        "*.md": "vscode.markdown.preview.editor",
    }
}

- Slow down CPU
sudo apt install cpupower-gui
In GUI, select min and max frequency, and governor "powersave"
