Vim

1. Basic config

~/.vimrc
syntax on
set number
set expandtab
set tabstop=4
set shiftwidth=4

2. Basic vim commands

vim$ :wq - write and quit

3. CXX autocomplete

$ sudo apt install curl
$ sudo apt install nodejs
$ sudo apt install npm
$ sudo apt install clangd

4. Install vim-plug (google)

5. coc nvim (google)

call plug#begin()
    Plug 'neoclide/coc.nvim', {'branch': 'release'}
call plug#end()

Install plugins:

vim$ :PlugInstall

6. coc nvim - Example Vim configuration. Copy to vimrc

7. coc nvim - using coc extension - coc-clangd

vim$ :CocInstall coc-clangd

8. More plugins

Plug 'jackguo380/vim-lsp-cxx-highlight'
Plug 'joshdick/onedark.vim'
Plug 'vim-airline/vim-airline'

File browser:

Plug 'preservim/nerdtree'

vim$ t: open file in new tab
vim$ ctrl+pgd: switch browser-like tab
