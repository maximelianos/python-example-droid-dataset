Ubuntu conda

=== install conda
wget -c https://repo.anaconda.com/miniconda/Miniconda3-py39_23.1.0-1-Linux-x86_64.sh
conda config --set auto_activate_base False

=== create environment + basic packages
conda create -n torch python=3.9 numpy pyyaml setuptools cmake pandas pillow scikit-image scikit-learn tqdm jupyter notebook matplotlib seaborn

=== heavy packages
pytorch torchvision opencv
conda-forge tensorboard / log learning progress
torchsummary / conda-forge pytorch-model-summary
